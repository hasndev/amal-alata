# موقع أمل العطاء - مؤسسة خيرية طبية

موقع إلكتروني احترافي لمؤسسة أمل العطاء الخيرية الطبية، مبني بتقنيات HTML, CSS, JavaScript, Bootstrap 5, و jQuery.

## المزايا الرئيسية

- ✅ تصميم RTL كامل للغة العربية
- ✅ هوية بصرية إسلامية أنيقة
- ✅ Bootstrap 5 RTL
- ✅ وضع ليلي/نهاري قابل للتبديل
- ✅ تصميم متجاوب (Responsive) للجوال والكمبيوتر
- ✅ نظام تبرع تفاعلي بثلاث خطوات
- ✅ خطوط عربية احترافية (Cairo & IBM Plex Sans Arabic)
- ✅ أيقونات Boxicons
- ✅ تأثيرات وحركات سلسة
- ✅ نموذج تواصل تفاعلي

## كيفية الاستخدام

1. **فتح الموقع محلياً:**
   - افتح ملف `index.html` مباشرة في المتصفح
   - أو استخدم Live Server في VS Code

2. **رفع الموقع على استضافة:**
   - ارفع جميع الملفات إلى سيرفر الاستضافة
   - تأكد من رفع جميع المجلدات (css, js, images)
   - الموقع لا يحتاج إلى قاعدة بيانات أو PHP

3. **التخصيص:**
   - عدّل معلومات الاتصال في جميع الصفحات
   - استبدل الشعار في مجلد `images`
   - عدّل الألوان في ملف `css/style.css`
   - عدّل المحتوى النصي في ملفات HTML

## ملاحظات مهمة

1. **نظام الدفع:** الموقع الحالي لا يحتوي على نظام دفع حقيقي. يجب ربطه بنظام دفع إلكتروني (مثل PayPal, Stripe, أو بوابات دفع محلية).

2. **قاعدة البيانات:** الموقع حالياً لا يحفظ البيانات. لحفظ التبرعات والاشتراكات، يجب إضافة Backend (PHP, Node.js, إلخ) وقاعدة بيانات.

3. **نموذج التواصل:** النموذج حالياً يعرض رسالة تأكيد فقط. يجب ربطه بخدمة إرسال البريد الإلكتروني.

4. **أوقات الصلاة:** الأوقات المعروضة هي أمثلة. يمكن ربط الموقع بـ API لأوقات الصلاة الحقيقية.
